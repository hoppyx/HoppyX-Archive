WaterMessage

Comme son nom l'indique, cette applet fait sortir des messages de l'eau. Très bien pour une présentation générale des rubrique de votre site. Vous pourrez télécharger cette applet en bas de page.

Mise en oeuvre

  • Téléchargez l'exemple, décompressez-le dans un dossier. Vous devriez trouver les fichiers watermessage.html et watermessage.class.
  • Ouvrez watermessage.html avec votre editeur, copiez le script de <!-- Début de l'applet --> à <!-- Fin de l'applet --> et collez-le dans votre page.
  • Veillez à placer avec votre page html le fichier watermessage.class.
  • Configurez les divers paramètres, ceux-ci sont documentés dans le code

Important

  • Uploadez toujours une applet en mode ASCII, faute de quoi le résultat sera un magnifique rectangle gris !
